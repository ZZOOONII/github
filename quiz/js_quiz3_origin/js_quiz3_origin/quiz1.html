Javascript Quiz

Q. 다음 중 삼항 연산자를 이용해 gender가 0이면 '남자', 0이 아니면 '여자'를 출력하는 코드로 옳은 것은? (gender 변수는 이미 선언되어 있다고 가정)

  1. console.log(gender == 0 : '남자' ? '여자');
  2. console.log(gender == 0 ? '남자' : '여자');
  3. console.log(gender == 0 : '여자' ? '남자');
  4. console.log(gender == 0 ? '여자' : '남자');

(해설) 자바스크립트에서 <> 연산자는 존재하지 않으며, while 반복문은 while (조건식) 형식으로 작성해주어야 한다.

Q. 다음 코드 실행 결과는?

  1. b
  2. a
    b
  3. b
    c
  4. a
    b
    c
  5. f

(해설) switch-case 문에서 case가 일치하더라도 break를 걸어주지 않으면 break를 만날 때까지 종료되지 않는다.

Q. 다음 중 while 반복문의 시작을 올바르게 작성한 것은?

  1. while (i <> 7)
  2. while (i !== 7)
  3. while (i =! 5) then
  4. while i != 7:

(해설) 자바스크립트에서 <> 연산자는 존재하지 않으며, while 반복문은 while (조건식) 형식으로 작성해주어야 한다.

Q. 다음 중 for 반복문의 시작을 올바르게 작성한 것은?

  1. for i = 1 to 5
  2. for (i <= 5; i++)
  3. for (i = 0; i <= 5)
  4. for (i = 0; i = 5; i++)

(해설) 자바스크립트에서 <> 연산자는 존재하지 않으며, for 반복문은 for (초기값; 조건식; 증감값) 형식으로 작성해주어야 한다.
