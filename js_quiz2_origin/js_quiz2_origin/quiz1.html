Javascript Quiz

Q1. 다음 중 js 파일을 불러오는 방법으로 옳은 것은?

  1. <script src="xxx.js">
  2. <script name="xxx.js">
  3. <script href="xxx.js">

(해설) 자바스크립트 파일은 script 태그의 src 속성에 불러온다.

Q2. 어떤 HTML 요소 안에 자바스크립트 문법을 작성해야 하는가?

  1. <js>
  2. <scripting>
  3. <javascript>
  4. <script>

(해설) 자바스크립트 파일은 script 태그안에서 작성할 수 있다.

Q3. 자바스크립트를 삽입하는 올바른 위치는?

  1. <head>와 <body> 모두 가능
  2. <head>
  3. <body>

(해설) 자바스크립트 파일은 <head>와 <body> 태그 에서 모두 사용 가능하다.

Q4. 자바스크립트 주석으로 올바른 것을 모두 고르시오.

  1. //
  2. <!-- -->
  3. #
  4. /* */

(해설) 자바스크립트 한 줄 주석은 // 이고, 여러 줄 주석은 /* */이다.

Q5. 자바스크립트 관례에 따른 올바른 변수명은 무엇인가?

  1. hello_world
  2. hello-world
  3. helloWorld
  4. HelloWorld

(해설) 자바스크립트 변수명은 camelCase 형식을 따라 작성하는 것이 관례이다.

Q6. 다음 중 변수를 올바르게 선언한 것은?

  1. v userId;
  2. const userId;
  3. let userId;
  4. string userId;

(해설) 변수 선언 키워드에는 <var>, <let>, <const>가 있으며, <const>는 선언과 동시에 값 할당이 이뤄져야 한다.

Q7. boolean 변수를 선언하는 방식으로 옳은 것은?

  1. let isWater = True;
  2. let isWater = true;
  3. let isWater = 'True';
  4. let isWater = 'true';

(해설) 자바스크립트 boolean 값은 true 또는 false 이다.

Q8. 다음 중 변수에 값을 할당하는 연산자는?

  1. =
  2. ==
  3. ===
  4. -

(해설) =는 변수에 값을 저장하는 할당 연산자이다.

Q9. Javascript와 Java는 같은 언어다

  1. O
  2. X

(해설) Java와 Javascript는 이름이 유사할 뿐 완전히 다른 언어입니다. 자바스크립트가 개발될 시점에 자바가 인기있는 프로그래밍 언어였기 때문에 그 인기에 편승할 목적(마케팅 목적)으로 이름을 지은 것 뿐입니다.

Q10. 다음 코드 실행 결과는?

  1. SeoulMapo
  2. Seoul Mapo
  3. Seoul
  4. Mapo

(해설) + 연산자로 문자열을 더할 때 공백을 넣지 않는다.

Q11. 다음 대화창을 출력하는 함수는 무엇인가?

  1. alert('아이디는?')
  2. window('아이디는?')
  3. prompt('아이디는?')
  4. confirm('아이디는?')

(해설) prompt()는 사용자가 텍스트를 입력할 수 있도록 안내하는 선택적 메세지를 갖고 있는 대화 상자이다.

Q12. 다음 중 식별자로 가능한 것을 모두 고르시오.

(참고) 식별자란 이름을 붙일 대 사용하는 단어로 변수나 함수에 이름에 쓰입니다.
  1. 2user
  2. User
  3. USER3
  4. *USER3
  5. USER_
  6. let
  7. _
  8. *
  9. $
  10. new user
  11. new_user
  12. $new_user
  13. const

(해설) 식별자 규칙은 다음과 같습니다. 키워드 사용 불가, 숫자 시작 x, 특수기호는 _, $만 허용, 공백 문자 X

Q13. Javascript에서 함수를 만드는 방법으로 옳은 것은?

  1. function = myFunction()
  2. function myFunction()
  3. function:myFunction()
  4. def myFunction():

(해설) 자바스크립트에서 함수를 선언하는 방식은 명시적 함수 선언과 함수 표현식 두 가지가 있다.
명시적 함수 선언 방식으로는 function myFunction() 다음과 같이 함수를 정의하며, 함수 표현식으로는 const myFunction = function () {} 다음과 같이 정의할 수 있다.

Q14. Javascript에서 myFunction 함수를 호출하는 방법으로 옳은 것은?

  1. call function MyFunction()
  2. call myFunction()
  3. myFunction()

(해설) 자바스크립트에서 함수를 호출할 때 함수 이름과 괄호를 이용해서 호출한다.

Q15. Boolean(10 > 9) 코드가 반환하는 값은?

  1. true
  2. false
  3. NaN

(해설) 10은 9보다 크므로 참(true) 값이다.

Q16. 다음 중 결과가 다른 하나는?

  1. !true
  2. true && false
  3. true || false
  4. !!false && (3 > 5)

(해설) 1, 2, 4는 false 3번은 true값이다.

Q17. 다음 중 결과가 다른 하나는?

  1. 3 !== '3'
  2. (8 / '2') !== 4
  3. 0 === false
  4. ('1' + 2) === 12

(해설) ==, != 연산자는 두 피연산자의 타입이 다르더라도 값만 비교하지만
===, !== 연산자는 엄격한 동등 연산자로 피연산자의 값과 타입을 모두 비교한다.

Q18. 다음 중 if 문을 올바르게 작성한 것은 무엇인가?

  1. if (x == 5)
  2. if (x == 5):
  3. if (x == 5) then
  4. if x = 5

(해설) 조건문은 if (조건식)으로 작성한다.

Q19. 자바스크립트에서 "i가 7이 아닐 때"를 의미하는 코드로 올바른 것은?

  1. if (i <> 7)
  2. if (i !== 7)
  3. if (i =! 7) then
  4. if i != 7:

(해설) i가 7이 아닐 때이므로 동등 연산자 중에서 != 혹은 !== 연산자를 이용해야 하며, if (조건식) 형식으로 작성해주어야 한다.
